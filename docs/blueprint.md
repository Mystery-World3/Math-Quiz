# **App Name**: LearnScape

## Core Features:

- Student Role: Students can input their name, select their class from a scroll-down menu on the landing page, access class-specific learning materials, and complete digital interactive exercises.
- Teacher Role: Teachers can securely log in to a dashboard via a hidden option (click on the logo) to create, read, update, and delete (CRUD) questions and manage content for different classes. Teacher dashboard includes a view of student grades.
- Interactive Exercises: Students can complete interactive digital learning exercises with immediate feedback on their answers, highlighting correct and incorrect responses.
- Grade Display: After completing the exercises, students can view their final score along with a detailed correction indicating the correct answers.
- Completion Confirmation: A 'Completed' option at the bottom of the results page for students to confirm they have reviewed their scores and corrections.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke a sense of trust, intelligence, and focus, suitable for an educational platform.
- Background color: Very light blue (#E8EAF6), a desaturated tint of the primary color to create a calming, distraction-free environment for learning.
- Accent color: Yellow-orange (#FFB300), to provide contrast, create emphasis and highlight key interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif for a modern look that still provides a little warmth to the educational platform.
- Use clear, simple icons for navigation and interactive elements.
- Clean and intuitive layout with clear sections for content, exercises, and results. Responsive design to ensure accessibility on various devices.
- Subtle transitions and animations to enhance user experience, such as highlighting feedback on exercises.
